# Build Repositories page with search, refresh, and Load all

## Description

Build the Repositories page for browsing catalog entries, searching loaded data, refreshing, and explicitly loading all pages.

## Acceptance Criteria

- [x] Repository names are listed in the Repositories page.
- [x] Current-page search filters only repositories already loaded in the browser.
- [x] `Load all repositories` fetches all pages through the backend API before filtering.
- [x] The page supports refreshing the current list.
- [x] Empty and error states are visible.
- [x] Typecheck/lint passes.
- [x] Verify in browser using dev-browser skill.
